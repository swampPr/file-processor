Compress a PDF Document

Here you can upload a PDF document and have it compressed. You can choose between our default option and our aggressive compression option. Please note that compression may take a while, especially if the file is large, or if the aggressive option is enabled. Aggressive compression will lower image quality in the document and make images and text black and white. The default compression will maintain an even balance between file size and quality. Only files below 500mb can be processed.
